# Projet d'Analyse Statistique – GHS Nigeria

## Auteur
Ouattara Ousmane

## Contexte
Ce projet analyse les données de la General Household Survey (GHS, Wave 4) du Nigeria dans le cadre du cours de statistique.

## Objectifs
- Étudier la distribution du niveau d'éducation
- Comparer selon le sexe et la zone (urbain/rural)
- Analyser les disparités régionales
- Tester les relations statistiques (chi-deux, V de Cramér)

## Contenu du dossier

- `rapport.Rmd` : code du rapport
- `rapport.pdf` : version finale
- `scripts/` : scripts d’analyse
- `data/` : données utilisées
- `output/` : graphiques et résultats

## Méthodologie
- Nettoyage et préparation des données (dplyr)
- Analyse descriptive
- Tests statistiques (chi-deux, Kruskal-Wallis)
- Visualisations (ggplot2)

## Reproductibilité
Le rapport peut être reproduit en exécutant le fichier `rapport.Rmd`.
