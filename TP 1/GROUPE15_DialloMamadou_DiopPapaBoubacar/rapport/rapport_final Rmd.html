Analyse : Le code commente l’utilisation du test de Shapiro-Wilk pour vérifier la normalité de la variable s1q4 (Âge). Comme la taille de l’échantillon est supérieure à 5, le test est applicable. La ligne est commentée (#), donc elle n’est pas exécutée.

Interprétation :

Si exécuté, shapiro.test(data$s1q4) testerait l’hypothèse nulle : les données suivent une distribution normale.

Un p-value > 0,05 ⇒ on ne rejette pas l’hypothèse nulle ⇒ les âges peuvent être considérés comme normaux.

Un p-value ≤ 0,05 ⇒ on rejette l’hypothèse ⇒ distribution non normale.

En résumé, ce code est prévu pour vérifier si la variable Âge suit une loi normale, mais pour l’instant le test est désactivé. # Pyramide des âges

total <- nrow(data)

resultats_parente <- data %>%
  filter(!is.na(s1q3)) %>%
  count(s1q3, name = "effectif") %>%
  mutate(
    proportion = effectif / total,
    proportion_pct = scales::percent(proportion, accuracy = 0.1),
    IC_lower = sapply(effectif, function(x) scoreci(x, total, conf.level = 0.95)$conf.int[1]),
    IC_upper = sapply(effectif, function(x) scoreci(x, total, conf.level = 0.95)$conf.int[2]),
    IC_text = paste0("[", scales::percent(IC_lower, accuracy = 0.1), " ; ", 
                     scales::percent(IC_upper, accuracy = 0.1), "]")
  )

# Affichage du tableau
resultats_parente %>%
  select(s1q3, effectif, proportion_pct, IC_text) %>%
  rename(
    `Lien de parenté` = s1q3,
    Effectif = effectif,
    Proportion = proportion_pct,
    `IC 95%` = IC_text
  )

inner_join(…, data_2, by = “hhid”) : on associe les informations de zone (ou autres données) à chaque ménage.

tab_resume <- data_tab %>%
